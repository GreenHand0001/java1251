# 校园社团信息管理系统 java1251

## 项目概述

校园社团信息管理系统是一个基于Springboot和Vue的前后端不分离的Web应用，旨在方便校园内的社团管理、活动组织以及成员交流。

## 技术栈

- **后端：** Spring Boot
- **前端：** Vue2 + ElementUI
- **开发工具：** IDEA + VSCode
- **Node.js版本：** 14
- **数据库：** MySQL 5.7及以上
- **JDK版本：** 1.8

## 功能模块

### 系统角色

- **管理员：**
  - 个人中心
  - 学生管理
  - 社长管理
  - 社团分类管理
  - 社团信息管理
  - 加入社团审核
  - 社团成员管理
  - 社团活动审核
  - 活动报名管理
  - 系统管理

- **社长：**
  - 添加社团
  - 社团活动创建
  - 社团成员管理

- **学生：**
  - 加入社团
  - 查看社团信息
  - 参与社团活动

### 运行环境

- JDK 1.8
- MySQL 5.7及以上
- IDEA（后端开发）
- VSCode（前端开发）

## 部署步骤

1. 环境搭建：安装JDK 1.8，MySQL 5.7及以上，配置相应环境变量。
2. 项目拉取：克隆或下载项目源代码。
3. 数据库初始化：执行数据库脚本，创建必要的表和初始化数据。
4. 项目编译：使用IDEA和VSCode分别编译后端和前端代码。
5. 项目部署：将编译后的代码部署到服务器。
